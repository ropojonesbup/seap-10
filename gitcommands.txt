Create a local Git Repositiory
    git init

Link local repository to remote git 
    git remote add origin url

Check the state of the remote repository
    git remote -v

Commit your chnages to the repository. a permanent save to create a version of your source code
 git commit -m "information about version"

Global Configuration Setting for Git
    git config --global user.email "you@example.com"
    git config --global user.name "Your Name"

Push data in your local repositiory to your remote repository
    git push 

View branches 
    git branch --list

Create branch
    git branch nameofbranch

Switch to branch
    git checkout nameofbranch
